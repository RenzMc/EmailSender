# EmailSender Tools

Tools sederhana untuk mengirim email promosi ke banyak penerima secara cepat dan efisien.

## Tentang Tools

EmailSender adalah tools Python yang dirancang untuk membantu pengiriman email promosi ke banyak penerima sekaligus. Tools ini menggunakan SMTP Gmail dengan autentikasi App Password untuk keamanan yang lebih baik.

## Fitur Utama

- Interface CLI yang menarik dengan Rich
- ASCII art keren
- Konfigurasi Gmail (email + app password)
- Multiple pesan yang dapat dikustomisasi
- Pengiriman ke banyak email sekaligus
- Tampilan responsif untuk semua ukuran terminal
- Penanganan interupsi (Ctrl+C) yang elegan

## Cara Install

```bash
# Clone repository
git clone https://github.com/RenzMc/emailsender.git
cd emailsender

# Install dependencies
pip install -r requirements.txt

# Jalankan tools
python email_sender.py
```

## Cara Penggunaan

1. **Setting Tools**: Konfigurasi akun Gmail dan pesan promosi
2. **Run Tools**: Tentukan jumlah pengiriman dan email tujuan
3. **Exit**: Keluar dari aplikasi

## Persyaratan

- Python 3.6+
- Koneksi internet
- Akun Gmail dengan App Password yang sudah diaktifkan
